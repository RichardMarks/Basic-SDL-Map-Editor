# readme.txt
# Basic SDL Map Editor
# developed by Richard Marks <ccpsceo@gmail.com>

version 1.0

* supports a single map and a single tile set
* map may be any size as defined in the startup.cfg file
* tiles are placed using the mouse
* camera is panned using the arrow keys or wsad keys
* saves maps to a simple binary format
* loads maps previously saved for continuing editing
