# Basic SDL Map Editor
# developed by Richard Marks <ccpsceo@gmail.com>

The wiki website for BSME can be found here:

http://gitorious.org/projects/basic-sdl-map-editor/pages/Home

================================================================================

version 0.0.5

* supports a single map and a single tile set
* maps are 122 x 90 tiles in size..this can not be changed for now
* tiles are placed using the left mouse button and removed with the right mouse button
* camera will pan when you move the mouse to any of the four edges surrounding the view and hold CTRL
* makes use of a simple 3 layer binary map file format
